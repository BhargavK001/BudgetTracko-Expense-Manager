# BudgetTracko - Expense Manager (v1.0.0)

BudgetTracko is a comprehensive expense management application designed to help users track their finances, manage budgets, and visualize their spending habits.

## Tech Stack

*   **Backend:** Node.js, Express, MongoDB (Mongoose)
*   **Frontend:** React, Vite, Tailwind CSS
*   **Mobile:** React Native (Expo) - *Note: Documentation for mobile app is currently separate.*

## Documentation

*   [Backend Documentation](./app/backend/README.md) - API endpoints, setup, and configuration.
*   [Frontend Documentation](./app/frontend/README.md) - Component structure, setup, and UI details.

## Quick Start

To get the project running locally, you will need to set up both the backend and frontend services. Please refer to the respective documentation links above for detailed instructions.
